# bias-review

A skill (and Gem) that critically reviews a piece of web content — article, post, paper,
product page — for bias, conflicts of interest, source quality, and reliability. It scores six
dimensions on a traffic-light scale and writes up where a reader should apply scrutiny.

- [`SKILL.md`](./SKILL.md) — Claude skill. Uses `web_fetch` to read the page + web search to research it.
- [`GEM.md`](./GEM.md) — Gemini Gem version. Same logic, but Google Search is the only tool.

---

## How to use this

Since this is a Gem AI assisted review that has categories, but not clearly defined rubrics, you should treat this as informational, not authoritative. The purpose is to provide additional context you may not know about the content; to surface who's behind a piece, what they might gain, how well it's evidenced, and where it's worth slowing down — so you can read it critically and make your own call. 
 

The following sections give more detail on why treating this tool as authoritative is a bad idea:

### It inherits the model's biases
The review is generated by a language model, and the model's training shapes what it treats as "neutral", "loaded", "credible". Those judgements reflect the data and the era the model was trained on. When the tool flags "loaded language" or "proportionate to the evidence", it's measuring against an internal baseline that is itself a position.

### Model choice changes the result
A bigger or more recent model may generally research more carefully, weigh conflicting signals better, and hedge more appropriately. A smaller or older one will be more confident and more
wrong. The same content run through different models can land on different results.

### It leans toward finding problems
The purpose of the skill is hunting for issues, so it is primed to find them. Two traps in particular:
- **Search asymmetry.** A search for "[author] + criticism" turns up critics for almost anyone, and famous people generate far more searchable controversy than obscure ones.
- **Red creep.** Because problems are what it's looking for, "clean" can be under-credited. The skill explicitly tries to keep green reachable, but the pull exists.

### It's not academic, and it's not a verdict on truth
This is not peer review, fact-checking, or a source-credibility study. There's no methodology to cite, no inter-rater reliability, no appeal. 
A biased source can be perfectly accurate, and a clean-looking one can mislead. The ratings tell you *how hard to look*, not *what's true*.
